# rawevoked

Landing page for the GitHub Pages project site served at
`https://talikamuhib.github.io/rawevoked/`.

## Notes

- Internal links to log entries use **relative paths** (e.g. `posts/log-007.html`) — never a
  leading slash, since a leading `/` would break on the `/rawevoked/` project-site path.
- Social links live in the footer of [index.html](index.html).
