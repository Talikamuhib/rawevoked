Where to start

If you're brand new, ignore everything below except these three. One book to read slowly, one course to follow along with, one tool to install. That's a complete starting kit.

An Introduction to the Event-Related Potential Technique beginner

Steven J. Luck · MIT Press (2nd ed., 2014)

The gentlest serious entry point. Explains why EEG/ERP works, in plain language, before any heavy math. If you buy one book, buy this one.

book

MNE-Python official tutorials beginner

MNE-Python contributors · free, online

Narrative, in-order tutorials that take you from loading data to evoked responses, each with runnable code. Free and constantly updated. This is the software you'll actually use.

free

MNE-Python tool

open-source Python package

Install this and you can follow along with every tutorial on this site. The whole ecosystem (Autoreject, PyPREP) plugs into it.

tool
01

Books

The canon. These are the books working researchers actually cite and recommend. Read Luck first, then reach for Cohen when you want the mathematics of the analysis.

An Introduction to the Event-Related Potential Technique beginner

Steven J. Luck · MIT Press

The standard first book on ERPs. Conceptual, readable, and honest about the field's pitfalls. Start here.

book

Analyzing Neural Time Series Data: Theory and Practice intermediate

Mike X Cohen · MIT Press (2014)

The definitive guide to the math and code of EEG analysis — time, time-frequency, and synchronization methods — written to be understood without heavy formal math training. Every method comes with code. The book most people call their EEG bible.

book

The Oxford Handbook of ERP Components reference

Steven J. Luck & Emily S. Kappenman (Eds.) · Oxford

A reference, not a read-through. When you meet a specific component (P300, N170, MMN) and want the definitive summary, look it up here.

book

Electric Fields of the Brain: The Neurophysics of EEG advanced

Paul L. Nunez & Ramesh Srinivasan · Oxford

The deep physics of where the EEG signal comes from — volume conduction, source localization, the biophysics. Reach for it when you want to understand the signal at its root. Some physics background helps.

book

Rhythms of the Brain context

György Buzsáki · Oxford

Not an analysis manual — the big-picture story of why the brain oscillates at all. Read it for intuition about what alpha, theta, and gamma actually mean.

book
02

Video + courses

For when reading isn't enough and you want someone to walk you through it. Mike X Cohen is the name to know — over 100 hours of free lectures that pair directly with his book.

Mike X Cohen — YouTube channel beginner → advanced

free video lectures

Lectures on time series analysis, the Fourier transform, and EEG methods — many are excerpts from full courses and map onto Analyzing Neural Time Series Data chapter by chapter. Start here for video.

video

FieldTrip video lectures intermediate

Donders / SCCN workshop recordings · free

Full workshop recordings covering oscillations, source modelling, connectivity, and non-parametric statistics — including a Cohen lecture series. Dense but excellent.

video

MNE-Python video tutorials beginner

community playlists · free

Screen-recorded walk-throughs of the MNE pipeline. Good to watch alongside the official written tutorials when a step doesn't click from text alone.

video

EEG/ERP Analysis with Python and MNE beginner

Udemy · paid (often discounted)

A structured, hands-on intro course: install Python, preprocess in MNE, run ICA, do frequency and ERP analysis. Good if you want one guided path start to finish and don't mind paying.

video
03

Tools

The software. You can do everything on this site with the first one; the rest are things you'll meet as you go deeper or read other people's papers.

MNE-Python primary

Python · open source

The tool this whole site is built around. Loading, filtering, ICA, epoching, spectra, source estimation — all in one well-documented package.

tool

Autoreject add-on

Python · works with MNE

Automatically learns per-channel rejection thresholds instead of you guessing one. The modern way to clean epochs. (Covered in tutorial 5.)

tool

EEGLAB alternative

MATLAB · open source

The big MATLAB toolbox. You don't need it if you're in Python, but half the literature and many tutorials use it, so it helps to recognise it.

tool

FieldTrip alternative

MATLAB · open source

The other major MATLAB toolbox, strong on connectivity and source analysis. Same note as EEGLAB — good to know it exists.

tool
04

Free docs + guides

Everything here is free and online. When a concept on this site is fuzzy, one of these usually has the clearest explanation.

MNE-Python tutorials + glossary essential

mne.tools · free

The single most useful free resource. In-order tutorials with real code and expected output, plus a glossary that defines the jargon.

free

Neurodesk handy

browser-based analysis environment · free

A ready-made neuroimaging environment you can run in the browser with datasets mounted — useful when you don't want to fight a local install.

free

intro-to-eeg course beginner

Alexander Enge · GitHub · free

An open, self-contained EEG-analysis course in Python and MNE. No prior Python required — a great structured companion to work through.

free

mikexcohen.com deep

Mike X Cohen · free lectures + book TOC

His site collects the free lectures, code, and workshop materials in one place. Bookmark it for when you want to go deep on a method.

free
05

Datasets

Where the data on this site comes from. All public, all free. The first two you can download from inside Python with one line.

EEG Motor Movement/Imagery (EEGBCI) beginner

PhysioNet · 109 subjects, 64 ch, 160 Hz

The dataset behind the alpha-waves tutorial. Includes eyes-open / eyes-closed baselines and motor-imagery runs. MNE downloads it for you.

free

MNE built-in datasets (sample, SSVEP, ERP CORE) beginner

mne.tools · one-line download

Small, clean example datasets built into MNE — perfect for learning a method before pointing it at bigger data. ERP CORE is purpose-built for classic ERP components.

free

OpenNeuro — EEG datasets all levels

openneuro.org · BIDS format

Hundreds of public EEG datasets in the standard BIDS layout — oddball, resting-state, food-word, and more. Where to go when you want real, varied data to practise on.
